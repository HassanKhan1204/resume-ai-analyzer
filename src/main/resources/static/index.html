Project Architecture

1. Upload Multipart REST API accepts PDF, DOCX, or text resumes.
2. Extract Apache Tika converts the file into analyzable text.
3. Analyze LLM-ready service returns missing skills, suggestions, and score.
4. Persist PostgreSQL stores metadata and analysis history.
